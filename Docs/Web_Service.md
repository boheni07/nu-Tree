## **\[분야 3\] 웹 서비스 및 관리 대시보드 개발 가이드 (Web\_Service.md)**

### **1\. 주요 기능**

* **공간 시각화:** 수집된 가로수 위치를 지도(Map)상에 마커로 표시.  
* **수종별 필터링:** 마커 색상 또는 아이콘을 수종별로 차별화하여 직관적 파악 가능.  
* **상세 정보창:** 마커 클릭 시 원본 사진, 센서 데이터, 1·2차 산정 크기 정보 등 모든 메타데이터 출력.

### **2\. 핵심 기술 스택**

* **Frontend:** React / Vue.js \+ Antigravity Web Component  
* **Map Engine:** Mapbox GL JS 또는 Kakao Maps API (국내 주소 체계 최적화)  
* **Data Visualization:** Chart.js (가로수 규격 분포 통계)

### **3\. UI/UX 구성 요소**

* **List View:** 필터링 및 검색 기능(수종, 식재일, 규격별 정렬)이 포함된 데이터 테이블.  
* **Map View:** \* **Clustering:** 광역 뷰에서 마커 밀집도 표시.  
  * **Overlay:** 특정 가로수 선택 시 레이어를 통해 수관폭 등을 지도상에 실척으로 가시화.  
* **Detail Modal:** 촬영 당시의 센서값(자이로스코프 기울기 등)을 시각적 인디케이터로 표현하여 데이터 신뢰도 확인.
